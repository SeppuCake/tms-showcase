# Tenant Management System (Showcase)

## Overview
A web application built for managing tenants, payments, and maintenance requests for residential properties.

## Purpose
To streamline property management operations and automate communication with tenants.

## Key Features
- Tenant registration and records
- Billing and payment tracking
- Maintenance request system
- Automated notifications (SMS / WhatsApp)

## Tech Stack
- PHP
- MySQL
- JavaScript
- API integrations (SMS services)

## My Role
- Developed system architecture and database design
- Built backend logic for billing and tracking
- Integrated messaging APIs for notifications
- Implemented user interface and workflows

## Impact
- Reduced manual tracking of tenants and payments
- Improved response time for maintenance issues
- Automated tenant communication
